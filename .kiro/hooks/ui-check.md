---

请执行以下 UI 检查任务：

1. 确认开发服务器已启动，如未启动请提示用户手动启动
2. 使用 Playwright MCP 打开当前开发的页面
3. 截取页面截图
4. 根据 Figma 设计稿（2x 倍率），检查以下内容：
   - 布局尺寸是否正确（记住 Figma 数值需要 ÷2）
   - 间距是否符合设计（margin、padding、gap）
   - 字号、字重是否正确
   - 颜色值是否匹配
   - 圆角、边框是否正确
   - 元素对齐方式是否正确
5. 列出发现的差异，按优先级排序
6. 给出具体的修复代码建议
